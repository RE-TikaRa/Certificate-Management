# Certificate Management Website

This branch (`WebPage`) contains the static website for the Certificate Management project, designed to be hosted on GitHub Pages.

## Structure

- `index.html`: The main landing page.
- `style.css`: Stylesheet for the landing page.
- `logo.svg`: Project logo.
- `.nojekyll`: Bypasses Jekyll processing on GitHub Pages to ensure all files are served correctly.

## Deployment

This branch is configured to be deployed via GitHub Pages.
